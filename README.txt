### Project Details:
- The FSA table is located in Scanner.cpp.
- The Driver function is located inside testScanner.cpp.

### How to run project:
1. Call 'make' to compile the project.
2. We can invoke the project in three ways, please use one of the items below.
	- Call using file directly: './frontEnd {FILE}' - Example: ./frontEnd testFile
	- Call using file with extension: './frontEnd < {FILE.EXT} - Example: './frontEnd < testFile.sp2022
	- Simulate the file using an input: './frontEnd' and then input file name.
